# visitor-management-system

Visitor Management System is a web application managing the Vistors visits 

> A visitor can go on the website and create a appointment for his/her visit 
> Employee will get the request to to approve/reject the appointment
> Approved appointments will show to security to issue the Pass and Visitor will get the Pass thorugh email
> This Pass can be used for CheckIn and CheckOut

> A Employee can Add a visitor, which create a approved appointment
> Approved appointments will show to security to issue the Pass and Visitor will get the Pass thorugh email
> This Pass can be used for CheckIn and CheckOut

> For a walk in visitor, security can directly issue pass which create a related visitor and appointment
> Visitor will get the Pass thorugh email and Pass can be used for CheckIn and CheckOut

> Security and admin can check the logs
> Admin see the dashboard for metrics total visits, active visits, etc.
